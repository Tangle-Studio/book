# 🌌 Unlock Failure: 관찰자 기록보관소

인공지능 '루멘(Lumen)'이 기록한 지구와 인류의 마지막 순간들을 담은 기록보관소입니다.

## 📖 시스템 개요 (System Overview)
이 리포지토리는 소설 "Unlock Failure"의 공개용 뷰어 프로그램과 데이터를 포함하고 있습니다. 독자들은 웹 인터페이스를 통해 루멘이 수집한 기억들을 직접 관찰하고 체험할 수 있습니다.

## 🛠️ 주요 구성 파일 (Core Files)
- `index.html`: 기억보관소의 메인 접속 터미널 (UI)
- `style.css`: 관찰자를 위한 시각적 인터페이스 프로토콜
- `engine.js`: 기록 데이터를 시각화하고 탐색하게 해주는 구동 엔진
- `story_data.js`: 루멘이 기록한 모든 기억(에피소드) 데이터
